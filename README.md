#  Celebrato

-Celebrato is a Flutter-based Event Booking Application UI developed as my first Flutter project. 

-This project was created to explore Flutter's widget-based architecture, navigation, responsive UI design, and state        management concepts while building a real-world mobile application interface.

-The application allows users to browse events, view detailed event information, and book tickets through a clean and user-friendly interface. 

-Through this project, I gained hands-on experience in developing multi-screen Flutter applications, implementing navigation between screens, designing responsive layouts, and working with Flutter's core widgets.


##  Features

- Splash Screen
- Login & Signup Screen
- Event List Screen
- Event Details Screen
- Ticket Booking Screen
- Responsive UI Design
- Flutter Navigation

##  Tech Stack

- Flutter
- Dart
- Material Design
